# Web Development 1 - HW3 - Responsive Images
In this repository, we will be working with differet ways to implement and make images responsive on a webpage using HTML, CSS, and JavaScript. There are three main ways we will showcase responsive images, those three being the picture element, the img tag and srcset attribute, and implementing using a background image in CSS. These three ways to showcase responsive images are very important for us to be able to make our pages clean and smooth at any size, from mobile to desktop.
## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
