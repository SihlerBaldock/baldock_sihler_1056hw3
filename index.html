Working with Responsive Images

The Picture Element

The HTML picture element is a container used to specify multiple source elements for a specific img contained in it. When using the picture element, you are strictly telling browser which image to use on different viewport widths. This can be very useful when one is going in more of an artistic direction.

The img Tag and srcset Attribute

When using the img tag and srcset attribute, you are essentially serving a different sized copy of the same image depending on the user's viewport width. This is called resolution switching, and you can accomplish it using the srcset and sizes attributes. When you use the srcset and sizes attributes on an img element, you are giving the browser necessary information to make a decision what image to request and serve to the user.

The CSS Background Image

The CSS background image property can be used alongside media queries to serve different images at different screen sizes. This can be very helpful if the image being used is decorative or part of a layout, not important or crucial to the content.
